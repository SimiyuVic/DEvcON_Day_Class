# DEvcON_Day_Class

A full-stack MERN application built using React (Vite), Node.js, Express, MongoDB, and JWT authentication.

## Getting Started

### Clone the Repository
git clone https://github.com/SimiyuVic/DEvcON_Day_Class.git
cd DEvcON_Day_Class

## Install Dependencies

### Frontend
cd frontend
npm install

### Backend
cd backend
npm install

## Environment Variables

Create a `.env` file inside the backend folder:

PORT=5000
MONGO_URI=your_mongodb_connection_string
JWT_SECRETKEY=your_secret_key

##  Run the Project

### Backend
cd backend
nodemon index

### Frontend
cd frontend
npm run dev

## Tech Stack

Frontend: React (Vite), Bootstrap  
Backend: Node.js, Express, MongoDB, Mongoose, JWT

## Features

- Authentication (Login/Register)
- JWT Security
- MongoDB Integration
- REST API
- React Frontend
